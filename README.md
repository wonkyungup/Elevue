## Sigus
> This program was created to make ssh tunneling easier.

### Commonly run before developers and packages
```
> npm cache clean --force
> npm install
```

### Developer mode
```
# window
> npm run dev:win

# linux
> npm run dev:linux

# osx
> npm run dev:mac
```

### Package mode
```
# window
> npm run build:win

# linux
> npm run build:linux

# osx
> npm run build:mac
```
